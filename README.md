# imd-chart-demo

## development setup

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev
```
## production setup
```bash

# cd to project directory
$ docker build -t imd-nuxtjs-chart-demo .

$ docker run -it -p 3000:3000 imd-nuxtjs-chart-demo

http://localhost:3000
```
